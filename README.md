# AI Yoga Trainer

## Overview

AI Yoga Trainer is a real-time yoga pose detection application built using Python, OpenCV, and MediaPipe. The system detects yoga poses from a webcam feed, provides posture feedback, calculates accuracy scores, and tracks pose holding duration.

## Features

* Real-time webcam-based pose detection
* Tree Pose Detection
* Warrior Pose Detection
* Chair Pose Detection
* Cobra Pose Detection
* Pose Accuracy Calculation
* Live Feedback System
* Pose Hold Timer
* Skeleton Landmark Visualization

## Technologies Used

* Python
* OpenCV
* MediaPipe
* NumPy
